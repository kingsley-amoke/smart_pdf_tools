# Privacy Policy for PDF Smart Tools

Last updated: [Current Date]

## Overview

PDF Smart Tools ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we handle your information.

## Data Collection

We do NOT collect, store, or transmit any of your personal data or files.

## File Processing

- All PDF processing happens locally on your device
- Files are never uploaded to our servers
- Files are not stored permanently
- Temporary files are automatically deleted after processing

## Internet Connection

The app requires internet only for:

- Connecting to our processing server
- No personal data is transmitted
- File processing is done server-side but files are immediately deleted

## Permissions

The app requires these permissions:

- READ_EXTERNAL_STORAGE: To access files you select
- WRITE_EXTERNAL_STORAGE: To save processed files
- INTERNET: To connect to processing server

## Third-Party Services

We do not use any third-party analytics, tracking, or advertising services.

## Children's Privacy

Our app is suitable for all ages and does not collect any personal information.

## Changes to This Policy

We may update this policy. Changes will be posted in the app.
